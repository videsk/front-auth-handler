Class: exports

exports()

Class to handle access and refresh JWT tokens for renewal and check validation. This library provide you automatic check validation over time expiration and server. Also every 1 second check locally if access token was expired and try to renewal with a refresh token in case you have it doing a HTTP request to custom endpoints.
